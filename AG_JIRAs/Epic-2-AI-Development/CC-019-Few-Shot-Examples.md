## User Story

As an **AI engineer**, I want to **create few-shot examples from sample clusters** so that **the AI learns from high-quality exemplars**.

---

## Description

Extract and format the best questions from sample clusters as few-shot examples to include in prompts. Select examples that demonstrate excellent 3D alignment.

---

## Acceptance Criteria

- [ ] 10-15 exemplar questions selected
- [ ] Examples cover all 5 domains
- [ ] Each example shows clear 3D alignment
- [ ] Examples formatted for prompt inclusion
- [ ] Variety of question types represented
- [ ] Examples library created

---

## Example Selection Criteria

- Clear 3D alignment
- Appropriate difficulty
- Good stimulus utilization
- Diverse question types
- Different domains represented
